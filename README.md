# Data Engineer Portfolio

Pipeline de datos end-to-end construido con Python, AWS, PostgreSQL, dbt, Airflow y FastAPI.

## Stack tecnológico
- Python
- AWS S3
- PostgreSQL + MongoDB
- dbt
- Airflow
- FastAPI
- Docker

## Estructura del proyecto
- ingestion/ → código de ingesta de datos desde APIs
- transformation/ → modelos dbt para limpieza y transformación
- orchestration/ → DAGs de Airflow
- api/ → endpoints FastAPI
- dashboard/ → visualizaciones
- tests/ → tests del código
- docs/ → documentación de arquitectura
